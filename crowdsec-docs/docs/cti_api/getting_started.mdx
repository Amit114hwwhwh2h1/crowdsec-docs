## Getting an API Key

When logged on the console, you can create an API Key from the "CTI API" section :

![create api key](/img/crowdsec-console-api-key-creation.png)

## Accessing the API

You can query the API about a given IP from the command line:

```shell
curl -H "x-api-key: YOUR_API_KEY" https://cti.api.crowdsec.net/v1/smoke/185.7.214.104 | jq .

```
